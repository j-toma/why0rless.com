All Stanford CS majors take a 6-course core (in addition to specializing in a track):

Systems I: Programming Abstractions—a classic CS2 course
emphasizing common data abstractions and structures as
well as recursion.
Systems II: Computer Organization and Systems—a course
with the theme of taking students "from the hardware up to
the source code". Topics include machine architecture,
memory models, data representation, and elements of
compilation and concurrency.
Systems III: Principles of Computer Systems—the theme of
this course is "modern computer systems and networks".
Topics include processes and concurrency mechanics, file
systems, virtualization, networking and distributed systems.
Theory I: Mathematical Foundations of Computing—this is
a course on discrete math and automata. Topics include
logic, induction, proof techniques, sets, functions, relations,
and an introduction to automata and NP-completeness.
Theory II: Probability Theory for Computer Scientists—an introductory course meant to provide students with tools for
probabilistic analysis and modeling in computing. The
course also provides an introduction to Machine Learning.
Theory III: Data Structures and Algorithms—an advanced
course in the design and analysis of algorithms, presenting
various data models (e.g., algorithms for trees and graphs)
and a range of algorithmic techniques (e.g. greedy and
randomized algorithms, dynamic programming).

Quoted from robotics.stanford.edu/users/sahami/papers-dir/SIGCSE-2010.pdf
