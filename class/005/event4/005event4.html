Assessment Event 4: Compare and Contrast Universities

Instructions Example
  • 1. 选择两个大学
  • 2. 研究他们--保存你用的网站(地址和.html文件)
  • 3. 做一个6页的PPT
    • 1标题页，添加名字和学号
    • 至少4页信息的幻灯片
    • 最后有页结论需要附上参考文献
  • 4. 关于两个大学:
    • 位置：城市、气候、有没有公共交通等
    • 专业：读该大学可以选择的专业、课程、学生与教师的比例、班级规模
    • 可提供条件：学术支持，贷款，住房和食品
    • 课外活动：社团活动，运动，相关俱乐部
    • 入学要求：SAT/ACT分数、GPA、英语要求、推荐信、学费
